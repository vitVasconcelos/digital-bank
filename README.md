Digital Bank Landing Page

Uma landing page moderna para um banco digital fictício, desenvolvida com HTML5 e CSS3. O projeto apresenta um design limpo e elegante, com foco em organização visual, seções informativas e boa experiência em telas desktop.

✨ Sobre o Projeto

Este projeto foi desenvolvido para desafio do Frontend Mentor, e ele possui a proposta de simular a página inicial de um banco digital, destacando seus principais serviços, vantagens e artigos recentes.

A interface foi construída com estrutura semântica em HTML e estilização personalizada em CSS, utilizando variáveis de cores, Flexbox e Grid Layout para organizar os elementos da página.
